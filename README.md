# Seasonal-Seeds-Budget-Analysis
- Exploratory Data Analysis Project on Variety of seeds and their budgets of each season. Recommended user with 6 seeds, 3 vegetable and 3 flower seeds. Source data is taken from seeds based website.
- Tools used:
1. **Pandas** to generate data frames.
2. **Beautiful Soup** and **Selenium** for web scraping.
3. **Statistics** for basic EDA.
4. **Seaborn** and **Matplotlib** for Univariate, Bivariate and Multivariate Analysis.

- Recommended Seeds for each season based on the available data.
